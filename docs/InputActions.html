Input actions and axes

InputActions<TAction, TAxis> unifies keyboard, mouse, and gamepad into digital actions and analog axes. It supports multi-binding, action sets, per-action repeat, axis combine policies and smoothing, JSON load/merge/save, and rebind capture.

Quick start

// Define enums for your game
enum MyAction { Jump, Pause, Interact, NavigateDown }
enum MyAxis { MoveX, MoveY, LookX, LookY }

// Create and bind (batch to coalesce change notifications)
var input = new InputActions<MyAction, MyAxis>();
using (input.BeginBindingsBatch())
{
    input.BindAction(MyAction.Jump, new KeyChordBinding(new KeyChord(Key.Space, KeyboardModifiers.None)));
    input.BindAction(MyAction.Pause, new KeyChordBinding(new KeyChord(Key.Escape, KeyboardModifiers.None)));

    input.BindAxis(MyAxis.MoveX, new Composite2KeysAxisBinding(Key.A, Key.D, 1f));
    input.BindAxis(MyAxis.MoveY, new Composite2KeysAxisBinding(Key.S, Key.W, 1f));

    // Frame-rate independent smoothing for movement
    input.SetAxisSmoothingTimeConstant(MyAxis.MoveX, 0.10f);
    input.SetAxisSmoothingTimeConstant(MyAxis.MoveY, 0.10f);

    // UI-friendly repeat (for menus)
    input.SetRepeat(MyAction.NavigateDown, enabled: true, initialDelaySeconds: 0.35, repeatSeconds: 0.05);
}

// Per-frame
input.Update(kb, mods, mouse, pads, dt);

if (input.TryConsumePressed(MyAction.Pause))
    TogglePause();

var move = input.Get2D(MyAxis.MoveX, MyAxis.MoveY);
if (input.WasPressed(MyAction.Jump))
    Jump();

Core queries

  • bool IsDown(TAction action), bool WasPressed(TAction action), bool WasReleased(TAction action)
  • bool WasRepeated(TAction action), bool WasPressedOrRepeated(TAction action)
  • float Get(TAxis axis) (smoothed or raw if no smoothing)
  • float GetRaw(TAxis axis) (pre-smoothing, useful for tuning)
  • Vector2 Get2D(TAxis x, TAxis y)
  • double GetHoldSeconds(TAction action) (duration held in seconds)
  • bool TryConsumePressed(TAction action) (returns true once per press)
  • InputDevice GetLastSource(TAction), InputDevice GetLastSource(TAxis)

Binding management

  • BindAction(TAction, params IActionBinding[]), BindAxis(TAxis, params IAxisBinding[]) (duplicates ignored)
  • UnbindAction(TAction), UnbindAxis(TAxis)
  • ClearActionBindings(TAction), ClearAxisBindings(TAxis)
  • HasAction(TAction), HasAxis(TAxis)
  • BeginBindingsUpdate(), EndBindingsUpdate(), BeginBindingsBatch() (RAII) — coalesce OnBindingsChanged
  • OnBindingsChanged — fired once after a batch of edits

Action sets and enable/disable

  • SetActionSet(TAction, string? setName), EnableSet(string set, bool enabled)
  • EnableAction(TAction, bool), EnableAxis(TAxis, bool)

Per-action repeat

  • SetRepeat(TAction, bool enabled, double initialDelaySeconds, double repeatSeconds)
  • WasRepeated(TAction), WasPressedOrRepeated(TAction)
input.SetRepeat(MyAction.NavigateDown, true, 0.35, 0.05);
if (input.WasPressedOrRepeated(MyAction.NavigateDown))
    MoveCursor(1);

Axis combine and smoothing

  • SetAxisCombinePolicy(TAxis, AxisCombinePolicy) — MaxAbs, SumClamped, FirstNonZero
  • SetAxisSmoothing(TAxis, float alpha) — 1 = instant, lower = smoother
  • SetAxisSmoothingTimeConstant(TAxis, float tauSeconds) — frame-rate independent smoothing
// UI look axis: simple alpha smoothing
input.SetAxisSmoothing(MyAxis.LookX, 0.25f);

// Player move axis: consistent smoothing across FPS
input.SetAxisSmoothingTimeConstant(MyAxis.MoveX, 0.10f);

Hold-duration

  • GetHoldSeconds(TAction) — increases while held, resets on release/disable
if (input.IsDown(MyAction.Interact) && input.GetHoldSeconds(MyAction.Interact) >= 1.0)
    ShowChargeEffect();

Consume pressed edges

  • TryConsumePressed(TAction) — returns true once per edge (prevents double-handling)
if (input.TryConsumePressed(MyAction.Pause))
    TogglePauseMenu();

JSON load/merge/save

  • string ToJson(), void WriteJsonTo(Stream)
  • void LoadFromJson(string json), void LoadFromJson(string json, bool strictEnums)
  • void MergeFromJson(string json), void MergeFromJson(string json, bool strictEnums)
  • void MergeFromJson(string json, bool strictEnums, MergeMode mode) — Replace or Append (unique only)
  • string ExportDiffJson(InputActions<TAction, TAxis> defaults) — emit only differences
  • string DumpState(bool includeBindings = false) — human-readable snapshot for debugging
// Append user overrides without removing defaults
input.MergeFromJson(json, strictEnums: true, mode: InputActions<MyAction, MyAxis>.MergeMode.Append);

// Export minimal user override vs defaults
var diff = input.ExportDiffJson(defaults);

Rebind capture helper

Capture the first meaningful input from the current frame. Keyboard capture skips pure modifiers/locks.

  • InputActions.InputCapture.TryCaptureAction(..., out IActionBinding? binding)
  • InputActions.InputCapture.TryCaptureAxis(..., out IAxisBinding? binding)

Examples

Batched edits and repeat

using (input.BeginBindingsBatch())
{
    input.ClearActionBindings(MyAction.NavigateDown);
    input.BindAction(MyAction.NavigateDown, new KeyChordBinding(new KeyChord(Key.Down, KeyboardModifiers.None)));
    input.SetRepeat(MyAction.NavigateDown, true, 0.35, 0.05);
}

Raw vs smoothed readout (tuning)

float rawX = input.GetRaw(MyAxis.MoveX);
float smX  = input.Get(MyAxis.MoveX);
DebugDrawAxis("MoveX", rawX, smX);

Behavior notes

  • Per-frame edges: call queries during your Update; transient state resets each frame.
  • Focus changes: call ResetState() on focus loss/profile swap to clear held/transients.
  • dt spikes: internal smoothing clamps large dt spikes for stability.
  • Duplicates: binding helpers avoid adding identical bindings (per binding type parameters).

Troubleshooting

  • Actions don’t fire: ensure the action is enabled and its action set is active.
  • Axes feel inconsistent: prefer time-constant smoothing for movement to decouple from frame rate.
  • JSON merge issues: use strictEnums: true while migrating to catch typos in enum names.
  • Multiple handlers run: gate logic with TryConsumePressed(action) to process an edge once.
