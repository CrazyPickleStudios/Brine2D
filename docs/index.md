# Brine2D

A modern 2D game engine for .NET 10. Build fast, ship confidently.

Brine2D is designed for developers who want a pragmatic, high-performance 2D engine with a clean API, strong tooling, and a focus on shipping games. This landing page gives you the big picture and clear paths to the docs you need.

## Why Brine2D
- Lightweight runtime with predictable performance
- Idiomatic .NET 10 API with familiar patterns
- Modular architecture: use only what you need
- Editor-friendly: optimized for Visual Studio workflows
- Production-ready: stability, testability, and observability

## Modern Runtime Model
Brine2D embraces contemporary .NET practices familiar to many developers:
- Async-first design across initialization, content pipelines, and I/O
- Host-based setup with dependency injection and configuration
- Extensible services and middleware-style loop customization

Learn more in the [Gameplay Overview](./gameplay/overview.md) and [Runtime Overview](./runtime/overview.md).
